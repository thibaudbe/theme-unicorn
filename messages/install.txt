Thanks for installing Unicorn Theme, a dark custom UI theme for Sublime Text 3 optimized for better readability.
This theme supports the new Sublime Text 3 icons for file types out of the box!

How to activate?
================

- Open your User Settings Preferences file:
    Sublime Text -> Preferences -> Settings - User
- Add (or update) your theme entry to be:
    "theme": "Unicorn.sublime-theme"
- Restart your editor

For more information visit: https://github.com/Thibaudbe/unicorn-theme
